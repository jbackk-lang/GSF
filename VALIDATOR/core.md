# GSF/VALIDATOR — walidator struktury globalnego pola finansowego

Walidator GSF łączy cztery moduły:
TIMDR — ciągłość danych,
GIA — redukcja modeli,
FIELDCORE — tensor wpływów,
CORE-NODES — rdzenie deformacji.

Walidacja polega na:
- wykryciu przerwań ciągłości,
- sprawdzeniu, czy model nie ma nadmiaru,
- ocenie, czy tensor wpływów jest kompletny,
- potwierdzeniu, że rdzenie pola są zmapowane.

Wynik walidacji nie jest liczbowy — jest strukturalny.
Walidator mówi, czy pole globalnych finansów jest spójne.
