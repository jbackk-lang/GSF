# GSF/CORE-NODES — rdzenie globalnego pola finansowego

Rdzenie pola to trzy główne źródła deformacji tensorowej:
USA, UE, Chiny.

Każdy rdzeń jest węzłem, który generuje napięcia w polach:
kapitał, waluty, handel, surowce.

Rdzeń USA:
- dominacja dolara,
- rynek obligacji jako stabilizator,
- wysoki poziom długu jako źródło napięcia.

Rdzeń UE:
- fragmentacja strukturalna,
- wielowarstwowy system bankowy,
- brak jednolitego pola fiskalnego.

Rdzeń Chin:
- kontrolowany system walutowy,
- napięcia w sektorze nieruchomości,
- silne sprzężenie z rynkiem surowców.

Każdy rdzeń jest analizowany jako osobny tensor deformacji.
